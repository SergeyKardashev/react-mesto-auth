# Проект: Место

### Описание проекта

Учебный проект для отработки навыков программирования на javascript и создания разметки на html и css.

**Технологии**

В работе применены:

- html
- css
- js
- БЭМ
- ООП
- модульный подход
- асинхронность
- запросы к серверу
- webpack
- React
- iron butt
